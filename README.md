# Dotfiles

Dotfile reposotiry for bspwm plus a whole bunch of other programs

## Program configs featured
- `bspwm`
- `dunst`
- `fish` and optionally `oh-my-fish`
- `mpv`
- `neovim`
- `picom`
- `polybar`
- `ranger`
- `rofi`
- `sxhkd`
- `wpgtk`

## Bare requirements to replicate my setup
- `bspwm`
- `wpgtk`
- `picom`
- `polybar`
- `rofi`
